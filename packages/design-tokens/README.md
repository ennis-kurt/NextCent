# Personal Accountant AI Design Tokens

Source-of-truth visual tokens for the premium financial interface:

- Deep neutral surfaces for trust-heavy dashboards
- Muted teal as the primary decision accent
- Warm brass for analytical highlights
- Explicit semantic colors for positive, warning, and risk states
- Number-first typography with tabular support

The web app mirrors these tokens in `apps/web/app/globals.css` so the MVP remains self-contained.
